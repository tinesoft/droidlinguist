How it works?

In just 3 simple steps, you can begin using the tool.

Step 1 - Languages Selection

Select the source language in which your strings.xml file is written, and then, the target languages you want your application to be translated into.

Step 2 - Translator Selection

Select a translator engine for automatic pre-translations.

You can choose between the most popular translator engines out there: Microsoft Translator, Yandex Translate, and soon, Google Translate (the later is already implemented, but not available at the moment, as it requires a paid API key).

Or if you prefer, you can also choose No Translator at all, to handle the translations directly yourself.

Step 3 - Strings.xml Upload

Upload your strings.xml and let the tool do the rest

It will do the following:

  • Check the validity of your file and inform you of any error in the XML
  • Translate all strings in the target langages you chose, using the translator you chose

That is it! Review & Download

You can now review all your translations, in a nice and clear interface, that will assist you during the whole process.

Once done, just hit the button to get the strings.zip that will contain the translated strings.xml files, and extract it to the res/values folder of your Android project.
