Features

Automatic Translations

droidlinguist allows you to quickly translate your strings.xml by using the most popular translator engines out there: Microsoft Translator, Yandex Translate, and soon, Google Translate (the later is already implemented, but not available at the moment, as it requires a paid API key).

Multiple Languages

You can edit multiple languages simultaneously by just toggling on/off the target languages you want to see.

Translations Validation

droidlinguist provides the following validation tests for translations:

  • Punctuation validation detects differences in the terminating punctuation between source and translated phrases when this may be significant. For example, warns if the source phrase ends with an ellipsis, exclamation mark or question mark, and the translated phrase does not, and vice versa.
  • Place marker validation detects whether the same variables (like %$1d, %$2d) are used both in the source text and in the translation.

Plurals, Arrays Support

droidlinguist supports editing strings that may have different translations, according to the quantity of items concerned. These are called Plurals

droidlinguist also supports editing Array of Strings.

Translations Follow Up

droidlinguist gives you instant feedback on your translation progress.

At each moment, and for each target language, you can know:

  • how many strings have not been translated yet
  • how many strings have validation errors
  • how many strings have been translated, but not validated yet
  • how many strings have been validated
